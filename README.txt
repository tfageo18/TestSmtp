Il s'agit d'un script VBS qui permet de tester son smtp depuis une machine.

Utilisation :
	- Ouvrir le fichier testSmtp.vbs avec un éditeur de texte (exemple notepad)
	- Modifier les variables avec celle correspondant à son SMTP
	- Cliquer sur le fichier testSmtp.bat si vous recevez un mail votre smtp fonctionne si vous recevez un message d'erreur cela ne fonctionne pas.
